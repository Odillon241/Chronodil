# Script vidéo : Création et gestion des projets

**Durée totale estimée : 7-8 minutes**

---

## 1. Introduction (0:00 - 0:30)

**[Écran : Page des projets]**

Bonjour et bienvenue dans ce tutoriel sur la création et la gestion des projets dans Chronodil. Les projets permettent de regrouper des tâches et d'organiser des équipes autour d'objectifs communs.

Dans cette vidéo de 7 à 8 minutes, nous allons voir comment créer un projet, gérer son équipe, lier des tâches, et suivre sa progression.

Cette fonctionnalité est disponible pour les rôles MANAGER, HR et ADMIN.

**[Transition : Zoom sur le bouton "+ Nouveau projet"]**

---

## 2. Accès à la page Projets (0:30 - 0:45)

**[Écran : Menu latéral avec "Projets" mis en évidence]**

Pour accéder à la gestion des projets, cliquez sur **"Projets"** dans le menu latéral gauche.

**[Animation : Clic sur "Projets"]**

Vous arrivez sur la page principale des projets, qui affiche tous vos projets actifs avec leurs informations principales : nom, description, équipe, nombre de tâches, et progression.

**[Écran : Liste des projets]**

---

## 3. Création d'un projet (0:45 - 2:30)

### Accès au formulaire

**[Écran : Bouton "+ Nouveau projet" en haut à droite]**

Pour créer un nouveau projet, cliquez sur le bouton **"+ Nouveau projet"** en haut à droite de la page.

**[Animation : Clic sur le bouton]**

Un formulaire s'ouvre avec tous les champs nécessaires.

### Remplissage des informations

**[Écran : Formulaire de création de projet]**

Remplissez les informations du projet :

**Nom du projet** : C'est un champ obligatoire. Entrez un nom clair et descriptif, par exemple "Site Web Entreprise" ou "Application Mobile".

**[Animation : Texte tapé dans le champ "Nom"]**

**Description** : Bien que non obligatoire, il est fortement recommandé d'ajouter une description. Décrivez les objectifs du projet, le contexte, et les livrables attendus.

**[Animation : Texte tapé dans le champ "Description"]**

**Date de début** : Sélectionnez la date de lancement du projet. Cliquez sur le calendrier pour choisir une date.

**[Animation : Sélection d'une date dans le calendrier]**

**Date de fin** : Sélectionnez la date d'échéance prévue. C'est important pour le suivi et la planification.

**[Animation : Sélection d'une date de fin]**

**Statut** : Par défaut, un nouveau projet est créé avec le statut **"ACTIVE"**. Vous pouvez le changer en **"ARCHIVED"** plus tard lorsque le projet est terminé.

**Budget** : Ce champ est optionnel. Vous pouvez indiquer le budget alloué au projet si vous souhaitez suivre les coûts.

**[Animation : Saisie d'un montant dans le champ Budget]**

### Création

**[Écran : Bouton "Créer le projet" en bas du formulaire]**

Une fois tous les champs remplis, cliquez sur **"Créer le projet"**.

**[Animation : Clic sur le bouton]**

Le projet est créé et vous êtes redirigé vers sa page de détails.

---

## 4. Gestion de l'équipe projet (2:30 - 4:00)

**[Écran : Page de détails du projet avec onglet "Équipe"]**

Une fois le projet créé, vous devez y ajouter des membres. Cliquez sur l'onglet **"Équipe"** dans la page de détails du projet.

**[Animation : Clic sur l'onglet "Équipe"]**

### Ajouter un membre

**[Écran : Section "Membres de l'équipe" avec bouton "Ajouter un membre"]**

Cliquez sur le bouton **"Ajouter un membre"**.

**[Animation : Clic sur le bouton]**

Un dialogue s'ouvre avec une liste de tous les utilisateurs disponibles.

**[Écran : Dialogue avec liste d'utilisateurs]**

Sélectionnez l'utilisateur que vous souhaitez ajouter au projet.

**[Animation : Sélection d'un utilisateur]**

**Définir le rôle** : Choisissez le rôle de l'utilisateur dans le projet :
- **Membre** : L'utilisateur peut voir et travailler sur les tâches du projet
- **Chef de projet** : L'utilisateur peut gérer l'équipe et modifier les informations du projet

**[Animation : Sélection du rôle "Chef de projet"]**

Cliquez sur **"Ajouter au projet"**.

**[Animation : Clic sur "Ajouter au projet"]**

L'utilisateur apparaît maintenant dans la liste des membres de l'équipe.

### Retirer un membre

**[Écran : Liste des membres avec icône de suppression]**

Pour retirer un membre de l'équipe, cliquez sur l'icône **✖️** à côté de son nom.

**[Animation : Clic sur l'icône de suppression]**

Un dialogue de confirmation apparaît. Confirmez la suppression.

**[Animation : Confirmation]**

Le membre est retiré de l'équipe et n'aura plus accès aux tâches du projet.

---

## 5. Lier des tâches au projet (4:00 - 5:00)

**[Écran : Page de détails du projet avec onglet "Tâches"]**

Il existe plusieurs méthodes pour lier des tâches à votre projet.

### Méthode 1 : Depuis le projet

**[Animation : Clic sur l'onglet "Tâches"]**

Cliquez sur l'onglet **"Tâches"** dans la page de détails du projet.

**[Écran : Onglet Tâches avec bouton "+ Nouvelle tâche"]**

Cliquez sur **"+ Nouvelle tâche"**. Le projet est automatiquement pré-sélectionné dans le formulaire de création de tâche.

**[Animation : Clic sur "+ Nouvelle tâche"]**

C'est la méthode la plus simple pour créer des tâches directement liées au projet.

### Méthode 2 : Lors de la création de tâche

**[Écran : Formulaire de création de tâche depuis la page Tâches]**

Lorsque vous créez une tâche depuis la page principale "Tâches", vous pouvez sélectionner le projet dans le champ dédié.

**[Animation : Sélection d'un projet dans le menu déroulant]**

### Méthode 3 : Réaffecter une tâche existante

**[Écran : Page de détails d'une tâche]**

Pour lier une tâche existante à un projet, ouvrez la tâche et modifiez le champ **"Projet"**.

**[Animation : Modification du champ Projet]**

Enregistrez les modifications. La tâche est maintenant liée au projet.

---

## 6. Suivi de la progression (5:00 - 5:30)

**[Écran : Page de détails du projet avec statistiques]**

La page de détails du projet affiche plusieurs indicateurs de progression :

**Statistiques des tâches** :
- Nombre total de tâches
- Tâches à faire (TODO)
- Tâches en cours (IN_PROGRESS)
- Tâches terminées (DONE)

**Progression** : Un graphique ou une barre de progression montre l'avancement global du projet.

**Heures travaillées** : Le total des heures déclarées sur les tâches du projet.

**[Animation : Curseur survolant les différentes statistiques]**

Ces indicateurs vous permettent de suivre l'avancement du projet en temps réel.

---

## 7. Archiver un projet (5:30 - 6:00)

**[Écran : Page de détails du projet avec bouton "Paramètres"]**

Lorsqu'un projet est terminé, vous pouvez l'archiver. Cliquez sur **"⚙️ Paramètres"** ou sur l'icône de paramètres.

**[Animation : Clic sur "Paramètres"]**

Changez le statut de **"ACTIVE"** à **"ARCHIVED"**.

**[Animation : Changement du statut]**

Enregistrez les modifications.

**Effets de l'archivage** :
- Le projet n'apparaît plus dans la liste des projets actifs
- Les tâches restent accessibles pour consultation
- Les données sont conservées pour les rapports et l'historique

**Restaurer un projet** : Pour restaurer un projet archivé, changez simplement son statut de **"ARCHIVED"** à **"ACTIVE"**.

---

## 8. Permissions et accès (6:00 - 6:30)

**[Écran : Tableau des permissions affiché]**

Il est important de comprendre les permissions selon les rôles :

**EMPLOYEE** :
- Peut voir les projets dont il est membre
- Ne peut pas créer, modifier ou archiver des projets

**MANAGER, HR, ADMIN** :
- Peuvent créer de nouveaux projets
- Peuvent modifier les projets dont ils sont chefs
- Peuvent ajouter des membres
- Peuvent archiver des projets

**ADMIN uniquement** :
- Peut supprimer des projets

**[Animation : Curseur survolant le tableau]**

Ces permissions garantissent que seules les personnes autorisées peuvent gérer les projets.

## 9. Conseils et bonnes pratiques (6:30 - 7:00)

**[Écran : Liste de conseils]**

Voici quelques conseils pour optimiser la gestion de vos projets :

### Structure claire

**[Écran : Exemple de structure de projet]**

Organisez vos projets avec une structure claire :
- Utilisez des noms de projets explicites
- Ajoutez toujours une description détaillée
- Définissez des dates de début et de fin réalistes

### Gestion d'équipe

**[Écran : Exemple d'équipe projet]**

- Définissez clairement les rôles (Membre vs Chef de projet)
- Ajoutez tous les membres nécessaires dès le début
- Retirez les membres qui ne sont plus impliqués

### Suivi régulier

**[Écran : Statistiques du projet]**

- Consultez régulièrement les statistiques de progression
- Utilisez les indicateurs pour identifier les retards
- Archivez les projets terminés pour garder une liste propre
