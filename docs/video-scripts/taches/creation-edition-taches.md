# Script vidéo : Création et édition de tâches

**Durée totale estimée : 6-7 minutes**

---

## 1. Introduction (0:00 - 0:20)

**[Écran : Page des tâches]**

Bonjour et bienvenue dans ce tutoriel sur la création et l'édition de tâches dans Chronodil. Les tâches sont l'élément central de votre organisation du travail.

Dans cette vidéo de 6 à 7 minutes, nous allons voir comment créer une tâche complète, la modifier, et utiliser toutes les fonctionnalités disponibles.

**[Transition : Zoom sur le bouton "+"]**

---

## 2. Créer une nouvelle tâche (0:20 - 3:30)

### Accès au formulaire

**[Écran : Bouton "+" en haut à droite]**

Pour créer une nouvelle tâche, cliquez sur le bouton **"+"** en haut à droite de la page des tâches, quelle que soit la vue que vous utilisez.

**[Animation : Clic sur le bouton "+"]**

Un formulaire s'ouvre avec tous les champs nécessaires.

### Informations principales

**[Écran : Formulaire de création avec champs principaux]**

Remplissez les informations principales :

**Nom de la tâche** : C'est un champ obligatoire. Entrez un nom clair et descriptif, par exemple "Développer l'API de connexion" ou "Réunion client - Présentation produit".

**[Animation : Texte tapé dans le champ "Nom"]**

**Description** : Bien que non obligatoire, ajoutez une description détaillée pour clarifier les objectifs et le contexte de la tâche.

**[Animation : Texte tapé dans le champ "Description"]**

### Projet et assignation

**[Écran : Champs Projet et Assigné à]**

**Projet** : Sélectionnez le projet auquel cette tâche appartient. Si la tâche n'est liée à aucun projet, vous pouvez laisser "Aucun projet".

**[Animation : Sélection d'un projet dans le menu déroulant]**

**Assigné à** : Sélectionnez l'utilisateur responsable de cette tâche. Vous pouvez vous assigner la tâche ou l'assigner à un collègue.

**[Animation : Sélection d'un utilisateur]**

### Priorité et complexité

**[Écran : Champs Priorité et Complexité]**

**Priorité** : Choisissez le niveau de priorité :
- **LOW** : Priorité basse, peut être reportée
- **MEDIUM** : Priorité moyenne, normale
- **HIGH** : Priorité haute, importante
- **URGENT** : Priorité urgente, à traiter immédiatement

**[Animation : Sélection de "HIGH" dans le menu]**

**Complexité** : Indiquez la complexité de la tâche :
- **FAIBLE** : Tâche simple, rapide à réaliser
- **MOYEN** : Tâche de complexité moyenne
- **ÉLEVÉ** : Tâche complexe, nécessite du temps et de l'expertise

**[Animation : Sélection de "MOYEN"]**

### Dates et heures

**[Écran : Champs de dates et heures]**

**Date de début** : Sélectionnez la date à laquelle vous prévoyez de commencer cette tâche.

**[Animation : Sélection d'une date dans le calendrier]**

**Date d'échéance** : Sélectionnez la date limite pour terminer la tâche. C'est important pour le suivi et les rappels.

**[Animation : Sélection d'une date d'échéance]**

**Heures estimées** : Indiquez le nombre d'heures que vous estimez nécessaires pour compléter la tâche. Cela aide à la planification.

**[Animation : Saisie de "8" dans le champ]**

### Statut initial

**[Écran : Champ Statut]**

**Statut** : Par défaut, une nouvelle tâche est créée avec le statut **"TODO"**. Vous pouvez le changer en **"IN_PROGRESS"** si vous commencez immédiatement.

**[Animation : Sélection du statut]**

### Création

**[Écran : Bouton "Créer la tâche" en bas]**

Une fois tous les champs remplis, cliquez sur **"Créer la tâche"**.

**[Animation : Clic sur le bouton]**

La tâche est créée et apparaît immédiatement dans toutes les vues (Liste, Kanban, Calendrier, etc.).

---

## 3. Éditer une tâche existante (3:30 - 5:00)

### Ouvrir une tâche

**[Écran : Liste de tâches]**

Pour éditer une tâche, cliquez dessus dans n'importe quelle vue.

**[Animation : Clic sur une tâche dans la vue Liste]**

Un formulaire d'édition s'ouvre avec toutes les informations pré-remplies.

### Modifier les informations

**[Écran : Formulaire d'édition]**

Vous pouvez modifier tous les champs :
- Nom et description
- Projet et assignation
- Priorité et complexité
- Dates et heures
- Statut

**[Animation : Modification de plusieurs champs]**

Par exemple, changez le statut de "TODO" à "IN_PROGRESS" lorsque vous commencez à travailler sur la tâche.

**[Animation : Changement du statut]**

### Commentaires

**[Écran : Section Commentaires]**

Dans le formulaire d'édition, vous trouverez une section **"Commentaires"** en bas.

**[Animation : Scroll vers la section Commentaires]**

Ajoutez des commentaires pour :
- Noter vos progrès
- Poser des questions
- Mentionner des collègues avec @nom
- Partager des informations importantes

**[Animation : Texte tapé dans les commentaires avec mention @]**

Les commentaires sont visibles par tous les membres du projet et permettent une collaboration efficace.

### Enregistrer les modifications

**[Écran : Bouton "Enregistrer"]**

Une fois vos modifications effectuées, cliquez sur **"Enregistrer"**.

**[Animation : Clic sur "Enregistrer"]**

Les modifications sont sauvegardées et visibles immédiatement par tous les membres de l'équipe.

---

## 4. Actions rapides (5:00 - 5:30)

**[Écran : Menu d'actions sur une tâche]**

Chronodil offre plusieurs actions rapides sur les tâches :

### Changer le statut rapidement

**[Animation : Clic sur le menu d'actions]**

Dans la vue Liste ou Kanban, vous pouvez changer rapidement le statut d'une tâche via le menu d'actions ou en la glissant-déposant dans la vue Kanban.

**[Animation : Glisser-déposer dans Kanban]**

### Dupliquer une tâche

**[Écran : Option "Dupliquer" dans le menu]**

Si vous devez créer une tâche similaire, utilisez l'option **"Dupliquer"**. La nouvelle tâche sera créée avec les mêmes informations, que vous pourrez ensuite modifier.

**[Animation : Clic sur "Dupliquer"]**

### Supprimer une tâche

**[Écran : Option "Supprimer" dans le menu]**

Pour supprimer une tâche, utilisez l'option **"Supprimer"** dans le menu d'actions. Un dialogue de confirmation apparaît.

**[Animation : Clic sur "Supprimer" et confirmation]**

---

## 5. Bonnes pratiques (5:30 - 6:00)

**[Écran : Liste de conseils]**

Voici quelques conseils pour créer des tâches efficaces :

### Nommage clair

**[Écran : Exemples de bons et mauvais noms]**

Utilisez des noms de tâches clairs et actionnables :
- ✅ "Développer l'API de connexion"
- ❌ "API"

### Descriptions détaillées

**[Écran : Exemple de description complète]**

Ajoutez toujours une description qui explique :
- Les objectifs de la tâche
- Le contexte
- Les critères de réussite

### Dates réalistes

**[Écran : Calendrier avec dates]**

Définissez des dates de début et d'échéance réalistes. Utilisez les heures estimées pour vous aider à planifier.

### Priorisation

**[Écran : Tâches avec différentes priorités]**

Utilisez la priorité pour identifier les tâches importantes. N'abusez pas de "URGENT" - réservez-le aux vraies urgences.

2. Éditer une tâche pour mettre à jour son statut et ses informations
3. Utiliser les commentaires pour collaborer
4. Profiter des actions rapides pour gagner du temps
