# Script vidéo : Découverte des 5 vues de tâches

**Durée totale estimée : 8-9 minutes**

---

## 1. Introduction (0:00 - 0:30)

**[Écran : Page des tâches]**

Bonjour et bienvenue dans ce tutoriel sur les 5 vues de tâches dans Chronodil. Chronodil offre 5 modes de visualisation différents pour s'adapter à votre style de travail et à vos besoins.

Dans cette vidéo de 8 à 9 minutes, nous allons explorer chaque vue et comprendre quand utiliser l'une plutôt que l'autre.

**[Transition : Zoom sur les onglets de vues]**

---

## 2. Accès aux vues (0:30 - 0:45)

**[Écran : Page des tâches avec onglets de vues]**

Pour accéder à la gestion des tâches, cliquez sur **"Tâches"** dans le menu latéral.

**[Animation : Clic sur "Tâches"]**

En haut de la page, vous trouverez 5 onglets correspondant aux 5 vues disponibles :
- **Liste** : Vue détaillée en tableau
- **Kanban** : Vue par colonnes avec glisser-déposer
- **Calendrier** : Vue par dates
- **Gantt** : Diagramme de Gantt pour planification
- **Roadmap** : Vue stratégique des jalons

**[Animation : Curseur survolant chaque onglet]**

---

## 3. Vue Liste (0:45 - 2:00)

**[Écran : Vue Liste avec tableau]**

La **Vue Liste** est la vue par défaut. Elle affiche toutes vos tâches dans un tableau détaillé.

### Colonnes du tableau

**[Zoom sur les colonnes du tableau]**

Le tableau affiche plusieurs colonnes :
- **Nom** : Le nom de la tâche
- **Projet** : Le projet associé (avec pastille colorée)
- **Statut** : TODO, IN_PROGRESS, ou DONE
- **Priorité** : LOW, MEDIUM, HIGH, ou URGENT
- **Complexité** : FAIBLE, MOYEN, ou ÉLEVÉ
- **Assigné à** : L'utilisateur responsable
- **Date d'échéance** : La date limite
- **Actions** : Boutons pour éditer, supprimer, etc.

**[Animation : Curseur survolant chaque colonne]**

### Tri et filtrage

**[Écran : En-têtes de colonnes cliquables]**

Cliquez sur un en-tête de colonne pour trier les tâches selon ce critère. Par exemple, cliquez sur "Priorité" pour trier par priorité.

**[Animation : Clic sur l'en-tête "Priorité"]**

Utilisez les filtres en haut de la page pour affiner votre vue :
- **Statut** : Filtrer par TODO, IN_PROGRESS, DONE
- **Priorité** : Filtrer par niveau de priorité
- **Projet** : Filtrer par projet spécifique
- **Assigné** : Mes tâches, Équipe, ou Tous

**[Animation : Utilisation des filtres]**

### Recherche textuelle

**[Écran : Champ de recherche]**

Le champ de recherche en haut permet de rechercher dans :
- Le nom de la tâche
- La description
- Les commentaires
- Les tags

**[Animation : Texte tapé dans le champ de recherche]**

La recherche est instantanée et filtre les résultats en temps réel.

---

## 4. Vue Kanban (2:00 - 3:00)

**[Écran : Vue Kanban avec colonnes]**

La **Vue Kanban** organise vos tâches en colonnes selon leur statut : TODO, IN_PROGRESS, et DONE.

### Glisser-déposer

**[Animation : Tâche glissée d'une colonne à l'autre]**

La fonctionnalité principale de la vue Kanban est le **glisser-déposer**. Cliquez sur une carte de tâche, maintenez le clic, et déplacez-la vers une autre colonne pour changer son statut.

Par exemple, glissez une tâche de "TODO" vers "IN_PROGRESS" pour indiquer que vous commencez à travailler dessus.

**[Animation : Plusieurs tâches déplacées]**

### Informations sur les cartes

**[Zoom sur une carte de tâche]**

Chaque carte affiche :
- Le nom de la tâche
- Le projet (pastille colorée)
- La priorité (badge)
- La date d'échéance
- L'utilisateur assigné (avatar)

**[Animation : Curseur survolant une carte]**

### Avantages de la vue Kanban

**[Écran : Vue Kanban complète]**

La vue Kanban est idéale pour :
- Visualiser rapidement l'avancement de vos tâches
- Réorganiser votre travail en déplaçant les cartes
- Voir d'un coup d'œil ce qui est à faire, en cours, et terminé

---

## 5. Vue Calendrier (3:00 - 4:00)

**[Écran : Vue Calendrier mensuelle]**

La **Vue Calendrier** affiche vos tâches organisées par date d'échéance.

### Navigation dans le calendrier

**[Animation : Curseur cliquant sur les flèches de navigation]**

Utilisez les flèches en haut pour naviguer entre les mois. Vous pouvez également passer en vue hebdomadaire ou quotidienne.

**[Animation : Changement de vue (mensuel → hebdomadaire)]**

### Tâches sur le calendrier

**[Zoom sur une date avec tâches]**

Les tâches apparaissent sur leur date d'échéance. Si plusieurs tâches ont la même date, elles sont empilées.

**[Animation : Curseur survolant une tâche sur le calendrier]**

### Glisser-déposer pour changer la date

**[Animation : Tâche glissée vers une autre date]**

Comme dans la vue Kanban, vous pouvez glisser-déposer une tâche vers une autre date pour modifier sa date d'échéance.

**[Animation : Tâche déplacée]**

### Avantages de la vue Calendrier

**[Écran : Vue Calendrier avec plusieurs tâches]**

La vue Calendrier est parfaite pour :
- Planifier votre travail sur plusieurs semaines
- Identifier les périodes chargées
- Respecter les échéances en visualisant les dates

---

## 6. Vue Gantt (4:00 - 5:30)

**[Écran : Vue Gantt avec diagramme]**

La **Vue Gantt** affiche un diagramme de Gantt classique pour la planification de projets.

### Structure du Gantt

**[Zoom sur le diagramme]**

Le diagramme de Gantt montre :
- **À gauche** : Liste des tâches avec leurs informations
- **À droite** : Timeline avec les barres de tâches

**[Animation : Curseur survolant le diagramme]**

### Barres de tâches

**[Zoom sur une barre de tâche]**

Chaque tâche est représentée par une barre horizontale qui s'étend de sa date de début à sa date de fin. La longueur de la barre indique la durée de la tâche.

**[Animation : Curseur survolant différentes barres]**

### Dépendances

**[Écran : Tâches avec flèches de dépendance]**

Les dépendances entre tâches sont représentées par des flèches. Par exemple, si la tâche B dépend de la tâche A, une flèche relie A à B.

**[Animation : Curseur suivant une flèche de dépendance]**

### Zoom et navigation

**[Animation : Zoom avant/arrière sur le Gantt]**

Utilisez la molette de la souris ou les contrôles de zoom pour ajuster la période affichée. Vous pouvez zoomer sur une semaine, un mois, ou voir toute l'année.

**[Animation : Navigation horizontale]**

Faites glisser la timeline horizontalement pour naviguer dans le temps.

### Avantages de la vue Gantt

**[Écran : Vue Gantt complète]**

La vue Gantt est essentielle pour :
- Planifier des projets complexes
- Visualiser les dépendances entre tâches
- Identifier les chevauchements et les conflits
- Gérer les ressources et les échéances

---

## 7. Vue Roadmap (5:30 - 6:30)

**[Écran : Vue Roadmap]**

La **Vue Roadmap** offre une vue stratégique à long terme avec les jalons et les phases de projets.

### Structure de la Roadmap

**[Zoom sur la roadmap]**

La roadmap affiche :
- Les projets organisés par phases ou jalons
- Les dates importantes (jalons, livraisons)
- La progression globale

**[Animation : Curseur survolant les éléments]**

### Jalons

**[Zoom sur un jalon]**

Les jalons sont des points clés du projet, comme "Lancement", "Version Beta", ou "Livraison finale". Ils sont affichés avec des marqueurs spéciaux.

**[Animation : Curseur survolant un jalon]**

### Navigation temporelle

**[Animation : Navigation dans la roadmap]**

Comme dans la vue Gantt, vous pouvez zoomer et naviguer dans le temps pour voir différentes périodes.

### Avantages de la vue Roadmap

**[Écran : Vue Roadmap complète]**

La vue Roadmap est idéale pour :
- Présenter la vision stratégique d'un projet
- Communiquer les jalons importants
- Planifier à long terme
- Aligner les équipes sur les objectifs

---

## 8. Création et édition de tâches (6:30 - 7:00)

**[Écran : Bouton "+" dans n'importe quelle vue]**

Quelle que soit la vue que vous utilisez, vous pouvez créer une nouvelle tâche en cliquant sur le bouton **"+"** en haut à droite.

**[Animation : Clic sur "+" depuis différentes vues]**

Le formulaire de création s'ouvre. Remplissez les informations :
- Nom de la tâche
- Description
- Projet associé
- Priorité et complexité
- Date d'échéance
- Utilisateur assigné

**[Animation : Remplissage du formulaire]**

Pour éditer une tâche, cliquez dessus dans n'importe quelle vue. Le formulaire d'édition s'ouvre avec les informations pré-remplies.

---

## 9. Quelle vue utiliser ? (7:00 - 7:30)

**[Écran : Tableau comparatif des vues]**

Voici un guide rapide pour choisir la bonne vue :

**Vue Liste** : Pour un travail quotidien, trier et filtrer précisément, voir tous les détails.

**Vue Kanban** : Pour gérer votre workflow, réorganiser rapidement, voir l'avancement visuel.

**Vue Calendrier** : Pour planifier sur plusieurs semaines, respecter les échéances, identifier les périodes chargées.

**Vue Gantt** : Pour planifier des projets complexes, gérer les dépendances, optimiser les ressources.

**Vue Roadmap** : Pour la vision stratégique, communiquer les jalons, planifier à long terme.

**[Animation : Curseur survolant chaque vue dans le tableau]**

N'hésitez pas à changer de vue selon vos besoins du moment !

Parfait ! Vous connaissez maintenant les 5 vues de tâches dans Chronodil.

**Récapitulons** :
1. **Vue Liste** : Tableau détaillé avec tri et filtres
