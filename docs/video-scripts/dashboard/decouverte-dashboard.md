# Script vidéo : Découverte du Dashboard

**Durée totale estimée : 5-6 minutes**

---

## 1. Introduction (0:00 - 0:20)

**[Écran : Page de connexion puis Dashboard]**

Bonjour et bienvenue dans ce tutoriel sur le Dashboard de Chronodil. Le Dashboard est votre point d'entrée principal dans l'application. Il vous offre une vue d'ensemble complète de votre activité, de vos projets et de vos statistiques.

Dans cette vidéo de 5 à 6 minutes, nous allons explorer ensemble toutes les sections du Dashboard et comprendre comment utiliser ces informations pour optimiser votre travail quotidien.

**[Transition : Dashboard complet]**

---

## 2. Vue d'ensemble du Dashboard (0:20 - 1:00)

**[Écran : Dashboard complet avec toutes les sections visibles]**

Le Dashboard de Chronodil est organisé en plusieurs sections principales :

**En haut** : Vous trouverez un message de bienvenue personnalisé avec votre nom, ainsi que la date du jour.

**Juste en dessous** : Trois cartes de statistiques principales qui affichent :
- Le nombre de tâches à faire
- Le nombre de tâches en cours
- Le nombre de tâches terminées

Ces cartes affichent également une comparaison avec la période précédente, indiquée par une flèche verte ou rouge et un pourcentage.

**En dessous** : Des onglets vous permettent de naviguer entre différentes vues :
- Feuilles de temps RH
- Projets
- Tâches récentes

**[Animation : Curseur survolant chaque section]**

---

## 3. Statistiques des tâches (1:00 - 2:00)

**[Écran : Zoom sur les trois cartes de statistiques]**

Les trois cartes de statistiques en haut du Dashboard vous donnent un aperçu rapide de votre charge de travail.

### Carte "Tâches à faire"

**[Zoom sur la première carte]**

Cette carte affiche le nombre total de vos tâches avec le statut "TODO", c'est-à-dire les tâches que vous devez encore commencer. 

À droite, vous verrez une flèche et un pourcentage qui indiquent l'évolution par rapport à la période précédente. Une flèche verte vers le haut signifie que vous avez plus de tâches à faire, tandis qu'une flèche rouge vers le bas indique une diminution.

**[Animation : Curseur pointant vers la flèche et le pourcentage]**

### Carte "En cours"

**[Zoom sur la deuxième carte]**

Cette carte affiche le nombre de tâches que vous avez actuellement en cours, c'est-à-dire avec le statut "IN_PROGRESS". C'est un indicateur important de votre charge de travail immédiate.

### Carte "Terminées"

**[Zoom sur la troisième carte]**

Cette carte affiche le nombre de tâches que vous avez complétées, avec le statut "DONE". C'est un excellent indicateur de votre productivité.

**[Animation : Curseur survolant les trois cartes]**

Ces statistiques sont mises à jour en temps réel. Dès que vous modifiez le statut d'une tâche, les chiffres se mettent à jour automatiquement.

---

## 4. Onglet Feuilles de temps RH (2:00 - 2:30)

**[Écran : Onglet "Feuilles de temps RH" sélectionné]**

Cliquez sur l'onglet **"Feuilles de temps RH"** pour voir vos dernières feuilles de temps.

**[Animation : Clic sur l'onglet]**

Cette section affiche :
- Vos feuilles de temps les plus récentes
- Le statut de chaque feuille (Brouillon, En attente, Approuvé, etc.)
- Le total d'heures déclarées
- La semaine concernée

**[Animation : Curseur survolant une feuille de temps]**

En cliquant sur une feuille de temps, vous accédez directement à ses détails. C'est un moyen rapide de vérifier l'état de vos soumissions ou de continuer à travailler sur une feuille en brouillon.

---

## 5. Onglet Projets (2:30 - 3:00)

**[Écran : Onglet "Projets" sélectionné]**

Cliquez sur l'onglet **"Projets"** pour voir vos projets actifs.

**[Animation : Clic sur l'onglet]**

Cette section affiche :
- Les projets auxquels vous êtes affecté
- Le nombre de tâches par projet
- La progression de chaque projet
- Une pastille colorée pour identifier rapidement chaque projet

**[Animation : Curseur survolant un projet]**

En cliquant sur un projet, vous accédez directement à sa page de détails. C'est idéal pour un accès rapide à vos projets les plus importants.

---

## 6. Onglet Tâches récentes (3:00 - 3:30)

**[Écran : Onglet "Tâches récentes" sélectionné]**

Cliquez sur l'onglet **"Tâches récentes"** pour voir vos dernières tâches.

**[Animation : Clic sur l'onglet]**

Cette section affiche un tableau avec :
- Les 10 dernières tâches créées ou modifiées
- Le nom de la tâche
- Le projet associé (avec sa couleur)
- Le statut (TODO, IN_PROGRESS, DONE)
- La priorité (LOW, MEDIUM, HIGH, URGENT)
- La complexité (FAIBLE, MOYEN, ÉLEVÉ)

**[Animation : Curseur survolant une ligne du tableau]**

En cliquant sur une tâche, vous accédez directement à ses détails. C'est parfait pour reprendre rapidement une tâche sur laquelle vous travailliez.

---

## 7. Actions rapides (3:30 - 4:00)

**[Écran : Bouton "+" en haut à droite]**

Le Dashboard offre également des actions rapides pour gagner du temps.

### Créer une tâche rapidement

**[Zoom sur le bouton "+" en haut à droite]**

En haut à droite du Dashboard, vous trouverez un bouton **"+"** qui vous permet de créer rapidement une nouvelle tâche sans quitter le Dashboard.

**[Animation : Clic sur le bouton "+"]**

Un formulaire s'ouvre directement, vous permettant de créer une tâche en quelques clics.

### Navigation rapide

**[Écran : Dashboard avec liens cliquables]**

Tous les éléments du Dashboard sont cliquables :
- Cliquez sur une feuille de temps pour voir ses détails
- Cliquez sur un projet pour accéder à sa page
- Cliquez sur une tâche pour l'ouvrir

**[Animation : Curseur cliquant sur différents éléments]**

Cette navigation fluide vous permet de passer rapidement d'une vue d'ensemble à une action spécifique.

---

## 8. Mise à jour en temps réel (4:00 - 4:30)

**[Écran : Dashboard avec indicateur de mise à jour]**

Le Dashboard de Chronodil se met à jour automatiquement en temps réel. Cela signifie que :

- Si un collègue modifie une tâche partagée, vous verrez la modification apparaître immédiatement
- Si vous créez une nouvelle tâche dans un autre onglet, elle apparaîtra dans les statistiques
- Les notifications importantes s'affichent automatiquement

**[Animation : Simulation d'une mise à jour]**

Vous n'avez pas besoin de rafraîchir la page manuellement. Tout se synchronise automatiquement grâce à la technologie WebSocket intégrée à Chronodil.

---

## 9. Conseils d'utilisation (4:30 - 5:00)

**[Écran : Liste de conseils affichée]**

Voici quelques conseils pour tirer le meilleur parti de votre Dashboard :

### Vérifiez régulièrement vos statistiques

**[Zoom sur les cartes de statistiques]**

Prenez l'habitude de consulter vos statistiques chaque matin. Cela vous permet de :
- Identifier rapidement votre charge de travail
- Planifier votre journée en fonction des tâches à faire
- Suivre votre progression

### Utilisez les onglets pour naviguer

**[Animation : Curseur cliquant sur les différents onglets]**

Les onglets du Dashboard sont votre point d'entrée vers les fonctionnalités principales. N'hésitez pas à les utiliser pour accéder rapidement aux sections qui vous intéressent.

### Personnalisez votre vue

**[Écran : Dashboard avec explication]**

Bien que le Dashboard soit standardisé, vous pouvez utiliser les filtres et les vues disponibles dans chaque section pour adapter l'affichage à vos besoins.
