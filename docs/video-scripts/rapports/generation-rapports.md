# Script vidéo : Génération de rapports et analyses

**Durée totale estimée : 6-7 minutes**

---

## 1. Introduction (0:00 - 0:30)

**[Écran : Page des rapports]**

Bonjour et bienvenue dans ce tutoriel sur la génération de rapports et analyses dans Chronodil. Les rapports vous permettent d'analyser votre activité, celle de votre équipe, ou de l'organisation entière selon votre rôle.

Dans cette vidéo de 6 à 7 minutes, nous allons voir comment générer différents types de rapports, configurer les filtres, et exporter les données.

**[Transition : Zoom sur les types de rapports]**

---

## 2. Accès aux rapports (0:30 - 0:45)

**[Écran : Menu latéral avec "Rapports" mis en évidence]**

Pour accéder aux rapports, cliquez sur **"Rapports"** dans le menu latéral gauche.

**[Animation : Clic sur "Rapports"]**

Vous arrivez sur la page principale des rapports, qui affiche un résumé de votre activité et des options pour générer des rapports détaillés.

**[Écran : Page des rapports]**

---

## 3. Types de rapports disponibles (0:45 - 1:30)

**[Écran : Section avec les différents types de rapports]**

Chronodil propose plusieurs types de rapports selon votre rôle :

### Rapport personnel

**[Zoom sur "Rapport personnel"]**

**Pour les employés** : Vos statistiques individuelles
- Heures travaillées (semaine, mois, année)
- Tâches terminées
- Répartition par projet
- Historique des feuilles de temps

**[Animation : Curseur survolant les métriques]**

### Rapport d'équipe

**[Zoom sur "Rapport d'équipe"]**

**Pour les managers** : Statistiques de votre équipe
- Heures par collaborateur
- Tâches en retard
- Charge de travail (capacité vs planifié)
- Projets en cours

**[Animation : Curseur survolant les métriques d'équipe]**

### Rapport global

**[Zoom sur "Rapport global"]**

**Pour HR/ADMIN** : Vue d'ensemble de l'organisation
- Total heures par département
- Coûts par projet
- Analyse de la productivité
- Données de paie

**[Animation : Curseur survolant les métriques globales]**

---

## 4. Générer un rapport (1:30 - 3:30)

### Sélectionner le type

**[Écran : Boutons de sélection du type de rapport]**

Cliquez sur le type de rapport que vous souhaitez générer.

**[Animation : Clic sur "Rapport personnel"]**

### Configurer les filtres

**[Écran : Section de filtres]**

Une fois le type sélectionné, configurez les filtres :

**Période** : Sélectionnez la période d'analyse
- Semaine en cours
- Mois en cours
- Trimestre
- Année
- Période personnalisée (dates de début et fin)

**[Animation : Sélection d'une période]**

**Utilisateurs** : (Pour managers et RH)
- Tous les utilisateurs
- Mon équipe
- Par département
- Sélection manuelle

**[Animation : Sélection d'utilisateurs]**

**Projets** :
- Tous les projets
- Projets actifs uniquement
- Projets archivés
- Sélection de projets spécifiques

**[Animation : Sélection de projets]**

**Statuts** :
- Toutes les tâches
- Tâches TODO uniquement
- Tâches IN_PROGRESS
- Tâches DONE

**[Animation : Sélection de statuts]**

### Sélectionner les métriques

**[Écran : Liste de métriques avec cases à cocher]**

Cochez les indicateurs que vous souhaitez inclure dans le rapport :

- ✅ Heures totales
- ✅ Heures par projet
- ✅ Heures par type d'activité
- ✅ Nombre de tâches
- ✅ Tâches terminées vs en cours
- ✅ Répartition par priorité
- ✅ Taux de complétion

**[Animation : Coche de plusieurs métriques]**

### Générer le rapport

**[Écran : Bouton "Générer le rapport"]**

Une fois les filtres et métriques configurés, cliquez sur **"Générer le rapport"**.

**[Animation : Clic sur le bouton]**

Le rapport est généré et affiché à l'écran avec des graphiques et des tableaux.

## 6. Exporter un rapport (4:30 - 5:30)

**[Écran : Boutons d'export]**

Une fois le rapport généré, vous pouvez l'exporter dans différents formats.

### Format Excel

**[Animation : Clic sur "Exporter en Excel"]**

Cliquez sur **"Exporter en Excel"** pour télécharger le rapport au format `.xlsx`. Ce format est idéal pour :
- Analyser les données dans Excel
- Créer vos propres graphiques
- Partager avec des collègues

### Format PDF

**[Animation : Clic sur "Exporter en PDF"]**

Cliquez sur **"Exporter en PDF"** pour télécharger le rapport au format `.pdf`. Ce format est parfait pour :
- Archiver le rapport
- L'imprimer
- Le partager par email

### Format CSV

**[Animation : Clic sur "Exporter en CSV"]**

Cliquez sur **"Exporter en CSV"** pour télécharger les données brutes au format `.csv`. Ce format est utile pour :
- Importer dans d'autres outils
- Traiter les données avec des scripts
- Analyser avec des outils de BI

---

## 7. Rapports mensuels et templates (5:30 - 6:00)

**[Écran : Section "Rapports mensuels"]**

Chronodil propose également des rapports pré-configurés :

### Rapports mensuels

**[Animation : Clic sur "Rapports mensuels"]**

Accédez à la section **"Rapports mensuels"** pour voir les rapports générés automatiquement chaque mois. Ces rapports sont prêts à l'emploi et ne nécessitent pas de configuration.

### Templates de rapports

**[Animation : Clic sur "Templates"]**

La section **"Templates"** contient des modèles de rapports pré-configurés que vous pouvez utiliser comme point de départ. Sélectionnez un template, ajustez les filtres si nécessaire, et générez votre rapport.

---

## 8. Conseils d'utilisation (6:00 - 6:30)

**[Écran : Liste de conseils]**

Voici quelques conseils pour tirer le meilleur parti des rapports :

### Utilisez les périodes personnalisées

**[Écran : Sélection de période personnalisée]**

Pour des analyses spécifiques, utilisez les périodes personnalisées plutôt que les périodes prédéfinies. Cela vous donne plus de flexibilité.

### Exportez régulièrement

**[Écran : Boutons d'export]**

Exportez vos rapports régulièrement pour garder un historique. Les rapports exportés peuvent être archivés pour référence future.

### Comparez les périodes

**[Écran : Rapport avec comparaison]**

Utilisez les comparaisons entre périodes pour identifier les tendances et les évolutions dans votre activité ou celle de votre équipe.
